初识PostgreSQL

发表于 2016-08-08   |   分类于 DB

安装

安装环境

CentOS 6.5

安装系统默认版本

$ yum install postgresql-server

安装完成后执行

$ service postgresql initdb
$ chkconfig postgresql on

如果想要安装某一版本的(这里举例9.5)，

$ yum install http://yum.postgresql.org/9.5/redhat/rhel-7-x86_64/pgdg-redhat95-9.5-2.noarch.rpm
$ yum install postgresql95-server postgresql95-contrib

$ service postgresql-9.5 initdb
$ chkconfig postgresql-9.5 on service postgresql-9.5 start

查看服务已经启动

$ ll /etc/init.d/postgresql-9.5

进入PostgreSQL命令行界面

PostgreSQL安装完成后，会默认生成一个名为postgres的linux用户，一个名为postgres的数据库和一个名为postgres的数据库用户。这个postgres就是数据库的超级账号。

切换到linux的postgres用户

$ su - postgres

可以看到postgres的安装目录（这里以PostgreSQL9.5版本举例）

$ ll
total 4
drwx------ 4 postgres postgres 4096 Aug  5 09:58 9.5

进入PostgreSQL命令行界面

$ psql

psql (9.5.3)
Type "help" for help.

postgres=#

创建新用户和新数据库

进入PostgreSQL命令行后，创建一个名为jjtest、密码为123456的数据库用户

create user jjtest with password '123456';

为jjtest创建一个数据库：jjtestdb

create database jjtestdb owner jjtest;

把数据库jjtestdb的所有操作权限赋给jjtest

grant all privileges on database jjtestdb to jjtest;

退出PostgreSQL命令行界面

\q 或者 Ctrl+D

用户登录

创建完jjtest用户及其拥有的db后，尝试登录

psql -U jjtest -d jjtestdb -p 5432 -h 127.0.0.1

需要密码，会出现下面的输入密码提示

Password for user jjtest:

输入密码 123456 后，进入命令行界面，当前数据库为jjtestdb

psql (9.5.3)
Type "help" for help.

jjtestdb=>

登录参数

-h, --host=HOSTNAME      database server host or socket directory (default: "local socket")
-p, --port=PORT          database server port (default: "5432")
-U, --username=USERNAME  database user name (default: "root")
-d, --dbname=DBNAME      database name to connect to (default: "root")

PSQL命令

$ psql --help

PSQL命令行里的命令

进入数据库后，可以用 \? 命令查看psql提供的命令

postgres-# \?

常用数据库操作

创建一个表
create table my_first_table(
    a integer,
    b integer,
    c integer,
    description text,
    primary key(a,c)
);
插入一条记录
insert into my_first_table values(1,1,2,'my first table description');
查询表
select * from my_first_table;
更新数据
update my_first_table set c=3 where a=1;
删除数据
delete from my_first_table where a=1;
创建索引
create index idx_desc on my_first_table (b);
Drop表
drop table my_first_table
查看所有数据库
\l
切换到某个数据库
\c jjtestdb
查看当前数据库所有表
\dt
查看表schema
\d my_first_table
或
\d+ my_first_table
